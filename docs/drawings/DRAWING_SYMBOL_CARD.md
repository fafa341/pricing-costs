# Tarjeta de Referencia — Símbolos de Despiece
**Imprimir: A6 (148 × 105 mm), plastificar**

---

## Tipos de Material

| Marca | Tipo |
|-------|------|
| ☑ Plancha | Lámina plana (cortar a medida) |
| ☑ Perfil | Perfil cuadrado / rectangular / tubo estructural |
| ☑ Tubo | Tubo sanitario (1½", ¾", etc.) |

---

## Símbolos de Proceso / Acabado

| Símbolo | Significado |
|---------|-------------|
| `P1` | Pulido grano 180 (satinado suave) |
| `P2` | Pulido grano 400 (espejo / brillante) |
| `T4` | Terminación T4 (scotch-brite) |
| `⊙` | Cilindrado (doblar en cilindro) |
| `S` | Soldadura visible (esmerilado requerido) |
| `V` | Visor / vidrio incluido |
| `M` | Mecanismo (bisagra, desagüe, cierre) |
| `EXT` | Proceso externo (subcontrato) |

---

## Dimensiones

```
Plancha:   L × A × esp    (mm)
Perfil:    L × sección    (mm, ej: 400 × 30×30)
Tubo:      L × Ø          (mm)
Cilíndro:  Ø × H × esp   (mm)
```

---

## Ejemplo

```
Nombre: Costado izquierdo (×2)
Tipo: ☑ Plancha
esp: 1.5 mm   L: 600   A: 400
Símbolo: P1
```

---

*Si no sabes el símbolo, deja el campo en blanco y escribe una nota en "Nombre".*
